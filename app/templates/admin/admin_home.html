Page How to use
Add a new admin user There will be a table that shows all the existing admin users in the system. From that you will have to choose a new username and password. Both must be under 8 characters long. If you choose an ID username that already exists, the system will notify you.
Deactivate an admin user There is a drop down menu that will allow you to choose the admin user to deactivate. Deactivated admin users can be reactivated for later use.
Reactivate an admin user There is a drop down menu that will allow you to choose the admin user to reactivate.
Download database as CSV There is a drop down menu that will allow you to choose the game session to download. You can name the file and should be able to find the downloaded version on pythonanywhere's folder. It will be under the file /nvgame/app/data.
Add a new session There is a table which shows all of the game templates to start a game. Choose the appropriate template and decide whether the game is a paced game or not. Finally, create the session.
Start a new session There is a table which shows all of the game templates to start a game. Choose the appropriate template and decide whether the game is a paced game or not. Finally, create the session. This session must be started @ "start a new session screen" for students to see it.
Stop an ongoing session There is a table which shows all of the ongoing sessions. Select the session you would like to stop in the drop down menu.
Create a new game template Choose a distribution that has been created to be used for the game template. Give this template a description. Select the session you would like to stop in the drop down menu.
Add a csv game template Choose a distribution that has been created to be used for the game template. Give this template a description. Select the session you would like to stop in the drop down menu.
Remove an existing game template Choose a template to delete from the drop down menu and click the delete button
Add a new distribution Follow the instructions on the page itself.
